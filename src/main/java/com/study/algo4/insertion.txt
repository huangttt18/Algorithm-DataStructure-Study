每次都将当前位置的元素(cur)和之前位置的所有元素(pre)进行比较，

if cur < pre then swap(cur, pre)

可以理解为摸牌的时候，在cur后面的元素其实都是不存在的，也就是还没有摸上来的牌

因此我们只对cur及pre进行比较并排序

对于只有一张牌的时候，也就是当i == 0的时候，默认就是有序的，因此也不需要对0这张牌进行排序
